# dotfile configurations
|Folder|for|
|------|---|
|`./hypr`|[Hyprland](https://hyprland.org/) Compositor|
|`./waybar`|[Waybar](https://github.com/Alexays/Waybar),a status bar running under wayland|
|`./nixos`|[NixOS](https://nixos.org/) Configurations|
|`./alacritty`|[Alacritty](https://alacritty.org/) Configurations|

**waybar requisites**  
FiraCode Nerd Font  
Noto Sans CJK SC

**hyprland requisites**  
`hyprlauncher`: A program launcher  
`alacritty`: A gpu-accelerated terminal emulator  
`waybar`: A status bar  
`nm-applet`: A system-tray app for managing network connections (requires NetworkManager)  
`xrdb`: for merging configurations with Xwayland  
`blueman-applet`: Manages bluetooth connections  
`fcitx5`: Input method
